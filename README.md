## Install & Start

⚠️ Using [Yarn Package Manager](https://yarnpkg.com) is recommended over `npm`.

Clone this project. Install the dependencies.

```shell
cd <app-root-dir>
yarn install
```

Run the app.

```shell
yarn start
```

The app runs on port 3000 by default and can be checked at http://localhost:3000
